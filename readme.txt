= Open HubSpot =

This extension uses the tab features of Thunderbird to open HubSpot
inside a Thunderbird tab.
    
= How it Works =
    
This extension adds a button to Thunderbirds main tool bar as well as an item to the
Tools menu which will open up HubSpot in a new tab.

first use you should see the HubSpot Account login page.  When you login you
will be prompted to save your username and password
for the site.  This password can be removed from the Security -> Passwords
section of your preferences.
